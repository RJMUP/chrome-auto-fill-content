# chrome-auto-fill-content

一款谷歌浏览器插件，帮你自动填写网页表单。只需点点鼠标告诉它要填哪个输入框、填什么内容，以后每次打开那个网页，账号密码、工号、地址等信息统统自动填好，省去重复输入的麻烦。

## 安装流程

1、下载本插件压缩包，解压到你电脑上的任意位置，记住解压出来的文件夹在哪里。

2、打开谷歌浏览器，在地址栏输入 `chrome://extensions/` 后回车。

3、打开页面右上角的 **"开发者模式"** 开关（点一下变成蓝色就行）。

4、点击左上角出现的 **"加载已解压的扩展程序"** 按钮。

5、在弹出的窗口中，找到并选中第 1 步解压出来的那个**文件夹**，点"选择文件夹"。

6、列表中出现了 **Auto Fill Content** 且开关是打开的，说明安装成功，可以开始用了。

## 使用说明

### 1、打开配置页面

安装完成后，在浏览器右上角点击 **扩展程序图标**（拼图形状），在列表中找到 **Auto Fill Content** 并点击，即可打开配置页面。

### 2、添加填充规则

假设你每天上班都要登录公司 OA 系统，每次都要输入账号密码，很麻烦。你可以这样做：

**第一步：填写基本信息**

在配置页面顶部，你会看到两个输入框：

- **网页地址匹配**：把你想自动填充的那个网页地址贴进来。比如 `https://oa.yourcompany.com`。注意，只需要贴网址的前半部分就行，不用把完整的长链接贴进来。
- **规则名称**：给这条规则起个名字，方便以后识别，比如 "公司OA登录"。

**第二步：让插件知道要填什么**

现在你需要告诉插件：页面上有哪些输入框需要填，以及填什么内容。

你已经看到一个字段行了，上面有三个输入框。我们只需要关注后两个：

- **填充值**：先填这个。这就是你想自动填入的内容，比如你的工号 `zhangsan`。
- **CSS选择器**：这个不用手写，点击选择器输入框右侧的 **🎯 按钮**，插件会自动帮你搞定（详见下一步）。

**第三步：用 🎯 选取页面元素**

1. 点击某个字段行右侧的 **🎯 按钮**。
2. 插件会自动打开你刚才填写的目标网页，并进入选取模式（页面顶部会看到提示条）。
3. 把鼠标移动到你想自动填充的输入框上，会出现蓝色高亮框。
4. **直接点击**那个输入框，选择器就自动获取完成了。
5. 页面会自动切回配置页，你会发现选择器输入框和填充值都已经自动填好了。
6. 如果点错了或者不想选了，按键盘上的 **ESC** 键退出。

> 💡 **小技巧**：建议先在目标网页中把表单填好，再用 🎯 选取元素，这样填充值也会自动带过来，省去手写的功夫。

**第四步：添加更多字段**

如果一个页面有多个输入框要填（比如用户名 + 密码），点击 **"+ 添加字段"** 按钮，重复第三步和第四步，把每个输入框都选上。

**第五步：保存**

点击 **"保存规则"** 按钮，搞定！

### 3、享受自动填充

保存规则后，每次打开那个网页，插件就会自动帮你把表单填好。你只需要点击登录按钮就行了。

### 4、管理已有规则

在页面下半部分可以看到所有已保存的规则：

- **编辑**：修改规则内容。
- **删除**：删掉不再需要的规则。

### 5、备份与迁移

页面底部有三个按钮：

- **导出配置**：把所有规则保存为一个文件，可以备份到电脑上，也可以分享给同事。
- **导入配置**：把之前导出的文件导回来，或者导入同事分享给你的配置。
- **清除所有配置**：一键清空所有规则，谨慎使用。
